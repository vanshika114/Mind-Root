# рџЊї MindRoot

> **An offline-first personal growth companion for mindful productivity, reflective journaling, and habit building.**

MindRoot is an open-source web application designed to help people build healthier routines through daily reflection, mood tracking, habit management, and guided mindfulness practices.

Instead of juggling multiple apps for journaling, habits, breathing exercises, and self-reflection, MindRoot brings everything together in a clean and distraction-free experience.

Built with a privacy-first philosophy, the application is designed to work offline and store user data securely, allowing users to focus on personal growth without sacrificing ownership of their information.
